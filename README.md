# home-portal

Vue/Vite web portal for `jorisjonkers.dev`.

## What It Is

`home-portal` is the public portfolio and authenticated application launcher for
the JorisJonkers-dev stack. It integrates with `auth-api` for session state and
uses `@jorisjonkers-dev/vue-web-commons` for shared UI behavior.

## Local Use

```bash
pnpm install
pnpm test
pnpm dev
```
